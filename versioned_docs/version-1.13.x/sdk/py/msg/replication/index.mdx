## ReplicationSettings

```python
class ReplicationSettings(BaseModel)
```

Settings for creating a replication

<a id="reduct.msg.replication.ReplicationSettings.src_bucket"></a>

#### src_bucket

source bucket name

<a id="reduct.msg.replication.ReplicationSettings.dst_bucket"></a>

#### dst_bucket

destination bucket name

<a id="reduct.msg.replication.ReplicationSettings.dst_host"></a>

#### dst_host

url of the destination instance

<a id="reduct.msg.replication.ReplicationSettings.dst_token"></a>

#### dst_token

access token for the destination instance

<a id="reduct.msg.replication.ReplicationSettings.entries"></a>

#### entries

list of entries to replicate. If empty, all entries are replicated.
Wildcards are supported

<a id="reduct.msg.replication.ReplicationSettings.include"></a>

#### include

replicate only records with these labels

<a id="reduct.msg.replication.ReplicationSettings.exclude"></a>

#### exclude

exclude records with these labels

<a id="reduct.msg.replication.ReplicationSettings.each_s"></a>

#### each_s

replicate a record every S seconds

<a id="reduct.msg.replication.ReplicationSettings.each_n"></a>

#### each_n

replicate every Nth record
